# 🏯 Устав клана Arasaka

## I. Общие положения
1. Правила регулируют порядок взаимодействия и внутреннюю структуру клана Arasaka.
2. Их соблюдение является обязательным для всех участников без исключений.
3. Присоединяясь к клану, игрок подтверждает согласие с данными правилами.
4. **Никнейм участника в игре должен полностью совпадать с никнеймом в Discord.** (Пример: `mitsuk_i | макнагец`).
5. Игрок, являющийся участником любой организации на любом из серверов, где клан ведёт свою деятельность, не имеет права вступать в клан (исключения — только по решению руководства).

## II. Правила поведения и коммуникации
### 2.1. Коммуникация
1. Токсичное, оскорбительное или провокационное поведение запрещено.
2. Публичные конфликты запрещены. Все споры — через руководство или Совет.
3. Запрещены кражи, гриф и обман соклановцев.

### 2.2. Клановый склад
1. Все участники обязаны пополнять склад, выполняя задания.
2. Ресурсы используются исключительно в интересах клана.
3. Продажа или передача ресурсов третьим лицам запрещена.
4. **Крупные личные склады запрещены.**
5. Новички не имеют доступа к складу без поручений.
6. Личные сундуки допускаются только в специально выделенной зоне.

### 2.3. Постройки и территория
1. Главный архитектор отвечает за планирование и участки.
2. Каждый участник обязан иметь жильё на территории клана.
3. Все постройки считаются собственностью клана.
4. Снос и перестройка — только с разрешения Главы или Совета.
5. Все постройки должны соответствовать стилю города.

## 🏗️ Раздел: Фермы клана

### 2.4. Общая информация
1. Главный механик отвечает за фермы и их техническое состояние.
2. 40% ресурсов, добываемых на АФК-фермах, передаются на склад.
3. Продажа ресурсов ферм осуществляется только через клановый магазин.
4. Новички не могут использовать фермы без разрешения.
5. Личный фарм допускается только с уведомлением главного механика.

### Правила доступа и использования
* Вход разрешён членам клана; гостям — только с разрешения механика или главы.
* Запрещено приводить посторонних лиц без разрешения.
* Любое строительство или модернизация — только после согласования с главным механиком.
* Запрещено оставлять фермы включёнными без прогрузчика.
