# Система дней рождений

Уведомления о днях рождения участников публикуются в канале **🍰・дни-рождения**.

Чтобы записать свой день рождения, пропишите команду в **🤖・команды**:

## Доступные команды

| Команда | Описание |
|--------|----------|
| `/birthday-add <день> <месяц> [год]` | Добавить свою дату рождения |
| `/birthday-set <пользователь> <день> <месяц> [год]` | Установить/обновить дату рождения пользователя *(только для персонала)* |
| `/birthday-list` | Список дней рождения всех пользователей |
| `/birthday-remove <пользователь>` | Удалить день рождения пользователя *(только для персонала)* |
